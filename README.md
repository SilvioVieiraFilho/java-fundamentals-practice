# Java Fundamentals Practice

Este repositório contém exercícios práticos de Java com foco no desenvolvimento backend.  
Os exemplos foram desenvolvidos durante meus estudos e têm como objetivo consolidar conceitos fundamentais da linguagem.

## 📚 Conteúdos abordados

- Arrays
- Control Flow (if, switch-case)
- Collections (List, Set, Map)
- Exception Handling (try-catch, NullPointerException)
- Generics
- Lambda Expressions
- Functional Interfaces
- Method Reference (::)

## 🛠️ Tecnologias

- Java
- JDK 8+

## 📂 Estrutura do Projeto

java-fundamentals-practice/
├── arrays/
├── collections/
├── exceptions/
├── generics/
├── lambda/
├── control-flow/

## 🚀 Exemplos implementados

- Manipulação de listas com `removeIf`
- Uso de `Map` com `put`, `get` e `forEach`
- Uso de `Set` para evitar duplicados
- Tratamento de exceções como `NullPointerException` e `ArrayIndexOutOfBoundsException`
- Criação de classes genéricas (`Caixa<T>`)
- Uso de expressões lambda para simplificar código
- Implementação de interfaces funcionais
- Uso de method reference (`System.out::println`)

## 🎯 Objetivo

Consolidar conhecimentos em Java e evoluir para o desenvolvimento backend, aplicando boas práticas e código limpo.
